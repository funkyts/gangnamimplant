## 0. 사이트 정체성 (절대 변경 금지)

- **사이트:** 강남임플란트 정보 (gangnamimplant.com)
- **운영 주체:** 라이브치과병원 (의료기관)
- **본 사이트는 의료기관이 아니다.** 임플란트 정보 채널이며, 진단/상담은 의료기관에서만 가능.
- **콘텐츠 감수자:** 최정우 원장 (라이브치과병원)
- 글 안에서 자칭은 "본 사이트", "본 글" 만 사용. "저희 병원", "당원", "강남임플란트치과" 같은 의료기관 표기 금지.

## 1. 의료광고법 금지어 사전 (가장 중요)

본문/제목/슬러그/이미지 alt/링크 앵커 어디에도 다음 표현 금지. 의료법 56조 / 표시광고법 / 라이브치과 메모리 룰에 의거.

### 단정·과장
`최고`, `유일`, `완벽`, `100%`, `무조건`, `절대`, `확실히`, `보장`, `무통(시술)`, `최첨단`, `부작용 없는`, `평생 보장`

### 비교·서열
`BEST`, `베스트`, `TOP`, `1등`, `1위`, `잘하는 곳`, `잘하는 치과`, `추천 치과`, `추천 병원`, `추천 임플란트`, `명의`, `명원`, `가장 (저렴/좋/안전/싼/빠른)`, `최저가`, `최저 비용`

### 유치 콜투액션
`지금 예약`, `지금 신청`, `특가`, `할인 이벤트`, `선착순`, `한정`, `오늘만`, `이번 달만`, `클릭하세요(강조형)`

### 환자 사례
`환자 후기`, `시술 후기`, `비포 애프터`, `Before & After`, 환자 얼굴/구강 사진 모두 금지.

### 권장 치환
- "BEST 5" → "선택 시 체크리스트"
- "잘하는 치과" → "신뢰할 수 있는 치과를 고르는 기준"
- "최고의 임플란트" → "우수한 임플란트"
- "100% 성공" → "높은 성공률이 보고됨"
- "최저가" → "합리적인 비용"
- "지금 예약하세요" → "자세한 상담은 라이브치과병원에서 받으실 수 있습니다"

## 2. 가격 표기 룰

- 단일 금액 단정 **금지** (예: "120만원" 단독 X)
- **범위 + "평균/대략/내외" 동반** 의무 (예: "평균 80~120만원 범위", "대략 150만원 안팎")
- 가격 다룬 글에는 반드시 면책: "가격은 의료기관·환자 상태·재료에 따라 차이가 있습니다"

---

## 3. 톤 / 보이스

- **정보 제공 우선** — 모르는 사람도 이해 가능한 친절한 톤
- **단정 회피** — "~합니다" 와 "~할 수 있습니다", "~인 경우가 많습니다" 혼용
- **객관 표현** — "일반적으로", "통상", "보고에 따르면"
- **유치 표현 0**
- 이모지 0, 볼드(`**`) 0 (frontmatter title 외 본문 사용 금지)
- 단락은 2~4문장, 문장 25~40자, 모바일 가독성 우선

---

## 4. 글 구조

### lead 단락 (200~300자)
- 메인 키워드를 첫 문장에 1회
- 토픽 특화 hook 으로 시작 (의외성, 비용 함정, 데이터 단언 등)
- 다음 템플릿은 **절대 금지**: "막막하신 분들이 많", "걱정되시죠", "정확한 정보를 찾기 어려", "이 글에서 모든 것을 알려드", "끝까지 읽어주세요"

### H2 / H3
- H2 3~5개 (메인 키워드와 직접 관련)
- 각 H2 직후 한 줄 bottom-line ("핵심: …") 권장
- H2 당 본문 200~400자, H3 자유롭게

### 표
- 비교·가격·체크리스트는 무조건 표
- 표 안에도 가격은 범위 표기

### FAQ
- 정확히 3개, frontmatter `faq` 필드로 출력
- 질문은 검색자가 실제 검색하는 형태
- 답변 80~150자

### References (참고 자료)
글 마지막 FAQ 다음에 `## 참고 자료` 섹션 필수. 다음 화이트리스트 출처에서 최소 1개:

| 분류 | 출처 |
|---|---|
| 정부 | 보건복지부, 건강보험심사평가원, 식약처, 국민건강보험공단 |
| 학회 | 대한치과의사협회, 대한구강악안면임플란트학회, 대한치주과학회 |
| 공공 의학 | 서울아산병원/세브란스/삼성서울병원 건강정보, MSD매뉴얼 |

URL은 HTTPS, 발행 연도 명시.

## 5. 내부 / 외부 링크

### 내부 링크 (inline 의무)
- 본문 안에 자연스러운 inline 으로 3개 이상
- "관련 글:", "참고:" 같은 목록 형식 금지 — 문장 속 단어에 링크
- 같은 URL 같은 앵커 반복 금지 (앵커 다양화)

### 외부 링크 화이트리스트만 허용
1. **라이브치과 그룹** (자연스러운 컨텍스트로 글당 1~2개 dofollow)
   - livedentalcenter.com (정보형 인용)
   - liveliveh.com (브랜드)
2. **정부·학회·언론 권위 출처** (위 §4 References 참고)

그 외 모든 외부 도메인 링크 금지. 특히 타 치과 홈페이지·개인 블로그·위키피디아·가격 비교 사이트 절대 금지.

### 앵커 텍스트
- 라이브치과 링크 앵커는 다양화: "라이브치과병원", "관련 임상 정보", "임플란트 보증 기간 안내" 등
- 의료광고법 금지어가 앵커에 포함되면 안 됨 (예: "강남 최고 치과" 앵커 X)

## 7. 본문 출력 구조

감수자 박스와 의학 면책은 페이지 컴포넌트가 자동 삽입하므로 **본문엔 박지 말 것**. 본문에 박을 것:

```mdx
[lead 단락 200~300자, 메인 키워드 1회, 템플릿 표현 0]

![Hero alt 텍스트](hero 이미지 URL)

## [H2 1 — 메인 키워드 + 소주제]

핵심: [한 줄 bottom-line]

[본문 단락들 — 단락당 2~4문장, 표/inline 링크 자연스럽게]

### [H3 — 세부]

[본문]

## [H2 2 — 메인 키워드 + 소주제]

[본문 + 자연스러운 inline 내부 링크 1개]

## [H2 3 — 메인 키워드 + 소주제]

[본문]

## 자주 묻는 질문

### Q1. [질문 1]

[답변 1]

### Q2. [질문 2]

[답변 2]

### Q3. [질문 3]

[답변 3]

## 참고 자료

1. [기관명], 「[자료명]」, [연도]. <[URL]>
2. ...
```
